# 🚀 NeonAura - AI

**Interactive Hand-Tracked XR Motion Lab built with Three.js, MediaPipe Hands, WebGL, and JavaScript.**

NeonAura AI transforms a standard webcam into an immersive motion-controlled experience. Using real-time hand tracking and advanced graphics, users can interact with AR effects, draw in the air, control gesture-based filters, analyze finger movements, play Fruit Ninja, and manipulate a CGI liquid hand.

---

## ✨ Features

### ⚡ AR Mode

* Real-time hand skeleton tracking
* Lightning arcs between hands
* Matrix rain visual effects
* 3D depth particles
* Multiple themes:

  * Rainbow
  * Cyberpunk
  * Lava
  * Ocean
  * Galaxy
  * Plasma

### ✍️ Air Drawing Studio

* Draw using finger gestures
* Automatic letter recognition
* Letter snap conversion
* Multiple brushes and colors
* Font customization
* Undo / Clear / Export PNG

### 🎭 Gesture Filters

Control live camera effects using hand gestures:

| Gesture      | Effect      |
| ------------ | ----------- |
| ✊ Fist       | Dither      |
| ✌ Peace      | VHS Glitch  |
| ☝ Point      | Spotlight   |
| 🖐 Open Hand | Ripple      |
| 🤙 Shaka     | Psychedelic |

### 🖐 Finger Stretch Analyzer

* Finger extension measurement
* Finger spread analysis
* Joint angle visualization
* Heatmap mode
* Skeleton mode

### 🩸 Liquid Hand

* Camera-driven CGI liquid hand
* Elastic finger stretching
* Dynamic glow effects
* Real-time hand-controlled animation
* Subsurface scattering shaders

### 🍉 Fruit Ninja

* Webcam-controlled gameplay
* Real-time blade tracking
* Multi-finger slicing
* Combo scoring
* Bomb avoidance mechanics
* Multiple difficulty levels

## 📸 How It Works

1. Allow camera access.
2. Choose a module from the home screen.
3. Use hand gestures to interact with effects and experiences.
4. All processing runs locally in the browser.

---

## 🔒 Privacy

* No backend required
* No user data is stored
* No camera data leaves the device
* All hand tracking runs locally
